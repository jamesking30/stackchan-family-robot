# 上游开源能力清单

| 项目 | 采用范围 | 许可证策略 |
|---|---|---|
| [m5stack/StackChan](https://github.com/m5stack/StackChan) | 官方 BSP、硬件驱动、OTA、设备协议 | MIT，锁定提交并保留许可 |
| [uezo/AIAvatarStackChan](https://github.com/uezo/AIAvatarStackChan) | 真机实时音频、表情、口型、摄像头 | MIT，作为固件应用基线 |
| [uezo/aiavatarkit](https://github.com/uezo/aiavatarkit) | Mac 实时对话网关、CharacterService、OpenClawTool | Apache-2.0，作为主机基线 |
| [uezo/chatmemory](https://github.com/uezo/chatmemory) | 长期记忆与 pgvector 搜索 | Apache-2.0，M3 引入 |
| [BrettKinny/dotty-stackchan](https://github.com/BrettKinny/dotty-stackchan) | 记忆命名空间、人物记忆与待审核模式参考 | MIT，仅复用设计模式 |
| [qua121/stackchan-simulator](https://github.com/qua121/stackchan-simulator) | 无硬件状态/动作验证 | Apache-2.0，M1 引入 |

无明确根目录许可证的仓库只作为接口调研来源，不复制代码。
